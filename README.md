## 📊 Overview

This project is an Excel-based dashboard analyzing call center data for 2023.

## 🔧 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* Formulas (IF, COUNT, VLOOKUP,XLOOKUP)

## 📈 Insights

* Customer-wise call performance
* Region-wise analysis (Cincinnati, Cleveland, Columbus)
* Call distribution across R01–R05
* Ranking analysis
